PC Build

I didn't have a good enough computer to run my CAD and slicer softwares, much less play games well, so I spent a few months learning all about PCs and how to build them. What I ended up with was this:

CPU - Ryzen 7 3700X

Motherboard - MSI B450 Tomahawk Max

RAM - G.Skill 16Gb 3600 MHz C16

Storage - HP EX920 256Gb NVMe

Graphics card - Sapphire Pulse 5700XT

Power supply - EVGA 750 Watt 80+ Bronze

Case - Fractal Design Meshify C

Extra case fan - Noctua NF A-14 PWM 140mm

I've made some minor tweaks and upgrades to this, so the pictures and link to the parts list below aren't 100% accurate, but I'll update this when I get another storage drive.
